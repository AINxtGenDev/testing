Quick Start Guide

Start Setup Auth Create Done!
1

Download & Install

Download the appropriate installer for your operating system from the releases page and run the installation wizard.

2

Open Your Project

Launch Auto-Claude UI and open a folder containing a git-initialized repository. The app works within git worktrees to protect your main branch.

3

Authenticate with Claude

Complete the OAuth authentication flow with your Claude Pro or Max account. This connects the AI agents to Anthropic's API.

4

Create Your First Task

Describe what you want to build in natural language. For example: "Create a REST API endpoint for user authentication with JWT tokens".

5

Watch the Magic Happen

Monitor autonomous agent progress in real-time through the Kanban board and agent terminals. Review changes when complete!

Configuration

Configure these environment variables in apps/backend/.env

Variable Required Purpose
CLAUDE_CODE_OAUTH_TOKEN Required OAuth authentication token for Claude
GRAPHITI_ENABLED Optional Enable memory/context retention features
AUTO_BUILD_MODEL Optional Override the default Claude model
GITLAB_TOKEN Optional GitLab authentication token
GITLAB_INSTANCE_URL Optional Custom GitLab server (defaults to gitlab.com)
LINEAR_API_KEY Optional Linear integration for task sync
